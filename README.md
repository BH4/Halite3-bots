# Halite3-bot
AI bot for Halite3 

This is the repository I used to produce a simple bot for Halite3 which just mostly reacted to specific situations. It was only made public after the competition ended. I ended up using 11 different revisions of my bot and got into the top 100 at one point. When I stopped updating my bot it dropped down to 365th place.
